# Workflow: 캐릭터 추가

## 목적
새 클레이 캐릭터를 도감에 등록한다.

## 필요한 것
- 캐릭터 이름
- 보관 장소 (장식장 / 7층집 / 텐트)
- 캐릭터 사진 (선택)
- 속성, 종족, 부과효과 정보 (선택)

## 실행 순서

### 1. 관리자 로그인
1. 앱 실행 후 우측 상단 자물쇠(🔒) 아이콘 클릭
2. 비밀번호 입력 (`.env.local`의 `ADMIN_PASSWORD` 값)
3. 자물쇠가 열린 모양(🔓)으로 바뀌면 완료

### 2. 캐릭터 추가 페이지 이동
1. 우측 상단 + 버튼 클릭 → `/characters/new` 페이지 이동

### 3. 정보 입력
1. 사진 영역 클릭 → 갤러리에서 사진 선택
2. 도감번호: 비워두면 자동 부여
3. 보관 장소 선택 (장식장 / 7층집 / 텐트)
4. 이름 입력 (필수)
5. 특장점 설명 입력
6. 체력 / 공격력 숫자 입력
7. 속성 버튼 탭해서 선택 (다중 가능)
8. 종족 드롭다운에서 1개 선택
9. 부과효과 버튼 탭해서 선택 (다중 가능)

### 4. 저장
- 하단 "캐릭터 추가" 버튼 클릭
- 성공 시 캐릭터 상세 페이지로 이동

## 오류 대처
- **사진 업로드 실패**: `D:\uploads\clay` 폴더가 있는지 확인
- **저장 실패 (401)**: 관리자 로그인이 풀렸을 수 있음 → 다시 로그인
- **저장 실패 (500)**: PostgreSQL 서버가 켜져 있는지 확인

## 관련 파일
- `app/characters/new/page.tsx` — 추가 페이지
- `components/CharacterForm.tsx` — 폼 컴포넌트
- `app/api/characters/route.ts` — POST API
- `app/api/upload/route.ts` — 이미지 업로드 API
